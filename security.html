Perguntas Frequentes

Tire suas dúvidas sobre a segurança do nosso sistema

Meus dados pessoais são coletados?

Não. Nosso sistema foi projetado para não coletar nenhum dado pessoal identificável. Não armazenamos IP, localização, informações do dispositivo ou qualquer outro dado que possa identificar o usuário.

Como é garantido o anonimato?

Utilizamos técnicas avançadas de anonimização, incluindo separação entre dados da denúncia e metadados, além de processos automatizados que removem qualquer informação potencialmente identificável.

Os dados são compartilhados com terceiros?

Não compartilhamos dados com terceiros. As informações são utilizadas exclusivamente para o propósito de melhorar o saneamento básico e são processadas de forma agregada e anônima.

Qual a duração do armazenamento dos dados?

Os dados são armazenados pelo tempo necessário para o processamento e análise estatística, seguindo os princípios da LGPD. Após este período, são automaticamente excluídos de forma segura.
